About

🚀 Task List is an application created as a personal exercise to master the technologies involved. In this case, I used Django, Python and Bootstrap in addition to the basic web technologies HTML, CSS and JS.

This app allows you to:

  • 📝 View tasks: The task list view allows you to see the name of pending tasks, the time to be performed and by clicking you can see the description associated with it.
  • ⏰ When the task's time arrives, a message will appear on the screen indicating that it's time to perform it!
  • ✏️ Edit tasks: The task editing view allows you to make any changes to the task attributes and then save the changes.
  • 🗑️ Delete tasks: The delete tasks view simply allows you to delete tasks and rearrange the list accordingly.
  • ✅ Completed Tasks: The completed tasks view allows you to see a list of tasks that you marked with the check button as completed tasks, and also allows you to reuse the completed task or delete it permanently.

I hope you enjoy this simple app, thank you! 😊
